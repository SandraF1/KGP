# Choosing a CSS framework

Rather than writing all the CSS myself, I decided to use a lightweight CSS preset.  I chose Pico, which has the following advantages:

## Easy installation

All it takes is an installation and an import.  I tried using a CDN which is a link in the head of the HTML file and does not require installation, but that didn't work for me.

## No frills

It is a basic but still tidy and attractive design.

## Clean and consistent

I liked the consistency afforded by a preset.  Pico in particular is very clear.

## Visibility

Pico has high contrast and lends itself well to accessibility requirements.
