# The first problem

I added a basic navigation bar to my learning platform, but the nav bar was lower for some tabs and higher for other tabs. This created an annoying jumping effect during navigation.

# Troubleshooting

I eventually noticed that the nav bar was consistent when the header was applied via the layout wrapper and imported to each page, rather than being applied globally via the main.tsx file.

# The next problem

I turned off horizontal scrolling via the CSS file, but the text was still obscured on the right hand side of the site.

# Troubleshooting

By adding a red border around all elements and a blue background to containers, I was able to deduce that the problem was with the flex. Specifically, I had to adapt the flex settings to allow text to wrap.

```
* {
  outline: 1px solid red;
}

.container {
  background: lightblue;
}
```

I used the following code to set the minimum width of the body to zero. Essentiallly, this means that the child container for the content has no set minimum and is allowed to shrink, which in turn wraps the text neatly to the viewport.

```
body > * {
    min-width: 0;
}
```
